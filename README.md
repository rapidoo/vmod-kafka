vmod-kafka
==========

vmod client for Kafka



test with UBUNTU 14.04, kafka kafka_2.9.2-0.8.1.1.tgz

depend from https://github.com/edenhill/librdkafka from master (29 sept 2014)


BUILD
-------

./configure VARNISHSRC=DIR [VMODDIR=DIR]
make
sudo make install
